# Bejamas Gatsby Starter

Highly opinionated Gatsby starter.

## Ingredients

* Gatsby v2 (beta)
* Styled Components

## Dev Ingredients

* Standard.js
* Prettier-standard
* Husky + lint-staged

## Components

* SEO
* Layout
